Towards a more open ESP8266

Hamish Coleman

ESP8266

  • $2 micro controller with wifi
  • However, not all open source

ESP8089

  • The same chip as the ESP8266, different firmware
  • Intended to be a low cost WIFI chip for low end tablets

ESP8089

  • Attaches via either SPI or SDIO
  • Runs as a dumb slave device
  • Host uploads firmware to start operation

Firmware

  • Expresif SDK - ~300k blob
  • ROM - 64K onboard
  • Firmware - 66K blob
